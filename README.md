📊 Project Title: E-Commerce Sales & Profit Performance Dashboard
Domain: Retail / E-commerce

Tools Used: Power BI, Power Query, DAX, MS Excel

📝 Project Overview
Developed an interactive Power BI dashboard to analyze sales performance for a US-based retail supermarket. The project involved transforming raw transactional data into actionable business insights to help stakeholders understand regional growth, product profitability, and customer segments.

🛠️ Key Steps Taken (The "How"):
Data Cleaning: Used Power Query to handle date formatting (Order Date/Ship Date), remove duplicates, and ensure data consistency across 18+ columns.

Data Modeling: Established relationships and created a dedicated Date Table for time-series analysis.

DAX Implementation: Calculated key business metrics including Total Sales, Total Profit, Profit Margin %, and Average Order Value (AOV).

Visualization: Designed a user-friendly interface featuring:

KPI Cards for high-level executive summaries.

Trend Lines to identify seasonal sales spikes.

Map Visuals to pinpoint high and low-performing states (e.g., Kentucky, California).

Slicers for dynamic filtering by Category (Furniture, Office Supplies) and Region.

💡 Business Insights (The "Why"):
Category Analysis: Identified that while 'Furniture' (like Bookcases) drives high revenue, 'Technology' often yields higher profit margins.

Shipping Efficiency: Analyzed the gap between Order Date and Ship Date to monitor logistical performance across different 'Ship Modes'.

Regional Growth: Discovered that the South Region showed specific growth patterns in the 'Consumer' segment.
